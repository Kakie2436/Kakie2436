# 基本思路

# 代码
```
#include<stdio.h>
#include<stdlib.h>

int main()
{
  int a[100];
  int n;
  scanf("%d",&n);
  for(int i=0;i<n;i++)
    scanf("%d",&a[i]);
  int temp;
  for(int i=0;i<n/2;i++)//若为奇数最中间的元素不会进入循坏，保持原位
  {
    temp = a[0+i];
    a[0+i] = a[n-1-i];
    a[n-1-i] = temp;
  }
  for(int i=0;i<n;i++)
    printf("%d ",a[i]);
  getchar();
  getchar();
}
```
